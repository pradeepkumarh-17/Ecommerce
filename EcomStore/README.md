# CloudCart - Premium Ecommerce Platform

CloudCart is a modern full-stack ecommerce web application built using Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript. The project delivers a premium online shopping experience with responsive UI, authentication system, cart management, wishlist, checkout flow, order tracking, and animated modern frontend design.

---

## 🚀 Features

* User Registration & Login
* Dynamic Product Listing
* Product Search & Filters
* Product Details Page
* Add to Cart System
* Wishlist System
* Checkout & Payment UI
* Order Tracking
* User Profile & Orders
* Responsive Premium UI
* MVC Architecture

---

### Frontend

## 📂 Project Structure

* Model → Java Classes
* View → JSP Pages
* Controller → Servlets
* DAO → Database Operations
* Database → MySQL

---

## ▶️ How to Run

1. Import project into Eclipse
2. Configure Apache Tomcat
3. Configure MySQL database
4. Run SQL scripts
5. Start Tomcat Server
6. Open:
   http://localhost:8080/EcomStore
